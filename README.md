# randomImage
URLSession을 이용한 네트워크 통신을 통해 이미지를 불러오는 샘플 앱

## Introduction
오픈소스 라이브러리를 사용하지 않고 URLSession의 작동원리 파악을 위해 진행
1. URLSession을 통해 API 데이터 불러오기
2. JSONDecoder를 이용해 데이터 파싱
3. 이미지 URL을 변환하여 UI에 표현

## Development Environmnet
* Swift
* Xcode
